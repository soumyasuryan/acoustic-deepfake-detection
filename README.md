# 🎙️ Multimodal Deepfake Audio Detection Pipeline (2D-CNN & DistilBERT Fusion)

An optimized, end-to-end deep learning framework designed to distinguish genuine human speech (*Bona Fide*) from synthetic audio clones (*Spoof*) generated by state-of-the-art Text-to-Speech (TTS) and Voice Conversion (VC) engines. 

This system upgrades a high-performance **2D-CNN Acoustic Channel** by integrating a **Semantic NLP Channel (Whisper-Tiny + DistilBERT)**. By cross-referencing acoustic signal anomalies with linguistic syntax tells through a **Late Fusion Decision Engine ($\alpha = 0.7$)**, the framework achieves exceptional robustness against unseen, out-of-distribution voice-cloning algorithms.

## 🔬 System Architecture & Pipeline

The pipeline transforms raw, variable-length audio waveforms into rigid, highly discriminative feature spaces across two parallel domains:

1. **Acoustic Channel:** Audio tracks are unified to a static 5-second duration at 16kHz. The system computes log-power Mel-Spectrogram feature grids using an $N_{FFT}=2048$ window and a $512$ hop length, outputting mathematical dimensions of `[1, 128, 157]`. These are classified by a hierarchical 2D-CNN with Batch Normalization and Dropout (0.3).
2. **Semantic Channel:** The audio is transcribed on the fly using **Whisper-Tiny**. The transcript is tokenized and passed through **DistilBERT** to extract a 768-dimensional contextual embedding vector, processed by a specialized linear classifier head.
3. **Late Fusion Block:** The final verdict is determined by combining the sigmoid probability scores of both networks:
$$\text{Fusion Score} = (0.7 \times P_{\text{acoustic}}) + (0.3 \times P_{\text{semantic}})$$
