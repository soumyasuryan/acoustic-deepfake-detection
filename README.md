# 🎙️ Acoustic Deepfake Detection using 2D-CNN & Spectral Feature Mapping

An optimized, end-to-end deep learning framework designed to distinguish genuine human speech (*Bona Fide*) from synthetic audio clones (*Spoof*) generated by state-of-the-art Text-to-Speech (TTS) and Voice Conversion (VC) engines. 

This model is trained on a highly balanced stream of the **ASVspoof 2019 Logical Access (LA)** dataset and achieves exceptional robustness against unseen, out-of-distribution voice-cloning algorithms.

---

## 🎯 Core Performance Highlights
* **Zero False Alarms:** Achieved **100% Macro Accuracy** on balanced in-distribution validation sets.
* **Bulletproof Security Boundary:** Maintained **99% Deepfake Precision** ($0.99$) when challenged with completely unseen generative speech models (OOD Test Partition).
* **Low False Positive Rate:** Triggered only **1 false alarm out of 200** real human voices, making it exceptionally safe for live biometric application layers.

---

## 🔬 System Architecture & Pipeline

The pipeline transforms raw, variable-length audio waveforms into rigid, highly discriminative time-frequency spatial matrices before executing a multi-layered 2D Convolutional neural grid.

1. **Acoustic Preprocessing:** Audio tracks are unified to a static 5-second duration (padding or trimming) at a 16kHz sampling rate.
2. **Feature Extraction:** Computes log-power Mel-Spectrogram feature grids using an $N_{FFT} = 2048$ window and a $512$ hop length, outputting clean mathematical dimensions of `[1, 128, 157]`.
3. **Deep Learning Classifier:** Passes spectral spatial maps through hierarchical 2D convolutional blocks integrated with 2D Batch Normalization, ReLU activations, and Dropout regularizers ($0.3$) to minimize structural overfitting.
